# 📊 Data Explorer Streamlit App

## Features
- Upload CSV
- Statistical Analysis (Mean, Median, Std, Variance)
- Histogram & Boxplot
- Insights
- PDF Report Download

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn
